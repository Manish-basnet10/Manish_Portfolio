# Image Placeholders

Drop your images here following this naming convention:

## /public/images/hero/
- `portrait.jpg` — Your portrait photo (recommended: at least 380×460px, object-fit: cover)

## /public/images/projects/
- `ecommerce.jpg`   — Screenshot of your E-Commerce project (16:9 recommended)
- `taskmanager.jpg` — Screenshot of your Task Manager project
- `devops.jpg`      — Screenshot of your DevOps Dashboard project
- `aichat.jpg`      — Screenshot of your AI Chat project
- (Add more matching the name in projectsData.js)

## /public/images/certifications/
- `aws-ccp.jpg`       — AWS Certified Cloud Practitioner certificate scan
- `meta-frontend.jpg` — Meta Front-End Developer certificate scan
- `google-data.jpg`   — Google Data Analytics certificate scan
- `docker.jpg`        — Docker certificate scan
- (Add more matching the image field in certificationsData.js)

## Guidelines
- Use JPEG or WebP for photos/scans; PNG for screenshots with text
- No spaces in filenames — use hyphens
- Forward slashes only in paths
- Recommended aspect ratios:
  - Hero portrait: ~1:1.2 (taller portrait)
  - Project screenshots: 16:9
  - Certificate scans: 4:3
